== Description ==

A chart plugin that uses ChartJS the open source chart JS project, allowing you to create any number of charts you can imagine.

== Installation ==

How To Install

1. Upload 'chart-plugin' folder to the '/wp-content/plugins/' directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Place '[chart id="1"]' in your content area.
